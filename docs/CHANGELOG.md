## [v0.1] — 2026-02-14

### Добавлено

- Первоначальная структура документации из `manual.md`
- **01-overview.md** — общее описание, стек, бренд, роли
- **02-user-stories.md** — пользовательские сценарии, UX-флоу, экраны, карта переходов
- **03-public.md** — публичная часть: каталог, поиск, популярное, ремиксы, шаринг
- **04-admin.md** — админка: теги, стили, курация, заказы, дропы
- **05-generation.md** — генерация: пайплайн, мутации, провайдеры, модерация
- **06-storage.md** — хранилище: модель данных, схема БД, файлы
- **07-orders-payments.md** — заказы, оплата ЮKassa, штрихкоды, лимиты
- **PLAN.md** — пошаговый план реализации с чеклистом
- **CHANGELOG.md** — этот файл

### Конвенции (зафиксировано)

- Enum: только строчные (string), не integer
- Деньги: integer в копейках
- HashID (6 символов, gem `hashid-rails`) вместо UUID для публичных ID
- slug для публичных URL сущностей (стили, теги, дизайны, дропы)

# 02. Пользовательские истории — FORMA

> Версия: v0.2 | Дата: 2026-02-15

## 1. Основные флоу

### Флоу A — "Каталог: выбрал — заказал"

```
Каталог → Коллекция/Стиль → Дизайн → Уровень исполнения (Core/Signature/Lux) →
  Наполнение → Оформление → Оплата → Заказ
```

**Экраны:** S1 → S-COL (или S7) → S9 → S8 → S10 → S11 → S12 → S17

Пользователь приходит в каталог, видит курированные коллекции и стили. Выбирает дизайн из каталога, видит 3 уровня исполнения (Core / Signature / Lux) с визуальными превью для каждого, выбирает наполнение, оплачивает.

### Флоу B — "С нуля: сгенерировал — заказал"

```
Каталог стилей → лайки → Создать → Генерация → 3 превью (основное + 2 мутации) →
  Докрутить/Выбрать → Наполнение → Уровень исполнения → Оплата → Заказ
```

**Экраны:** S1 → S2 → S3 → S4 → S5 → S8 → S9 → S10 → S11 → S12 → S17

Пользователь сам задает промпт и теги, получает 3 превью (основное + 2 мутации), докручивает до идеала. При заказе выбирает уровень исполнения — но без tier-специфичных визуальных превью (они доступны только для каталожных дизайнов, подготовленных куратором).

### Флоу C — "Увидел чужой дизайн → ремикснул"

```
Страница дизайна → Ремикс → Генерация (3 превью) → Докрутить → Заказ
```

**Экраны:** S7 → S6b → S4 → S5 → ...

### Флоу D — "Лимит → безлимит на период"

```
Генерация → лимит → экран покупки → ЮKassa → вернулся → генерирует дальше
```

**Экраны:** S3/S6b → L1 → S11 → (success) → назад в S3/S6b

### Флоу E — "Докрутка"

```
S5 → S6b → S4 → S5 (повторяем)
```

### Флоу F — "Коллекция → дизайн → заказ"

```
Каталог → Коллекция → Дизайн → Уровень → Наполнение → Заказ
```

**Экраны:** S1 → S-COL → S7 → S9 → S8 → S10 → S11 → S12

### Флоу G — "Дроп (лимитированная серия)"

```
Каталог / Рассылка / DeepLink → Дроп → Дизайн → Уровень → Наполнение → Заказ
```

**Экраны:** S1 / deeplink → S-DROP → S7 → S9 → S8 → S10 → S11 → S12

Лимитированная серия — тираж N штук с нумерацией экземпляров. Пользователь видит "Осталось X/30", при оформлении заказа ему присваивается номер.

## 2. Два пути к продукту

### Путь A — Каталог (курированные дизайны)

Дизайны подготовлены куратором через Training Pipeline (загрузка референсов → AI-анализ → курирование → генерация в 3 уровнях). Для каждого каталожного дизайна существуют визуальные превью для Core, Signature и Lux — пользователь видит, как выглядит блокнот в каждом исполнении.

**Характеристики:**
- 3 визуальных превью (Core / Signature / Lux) для каждого дизайна
- Объединение в коллекции (regular и limited)
- Лимитированные серии с нумерацией и ограниченным тиражом
- Дизайн — результат курирования, не пользовательской генерации

### Путь B — Генерация (пользовательские дизайны)

Пользователь сам создает дизайн через промпт + теги, получает 3 превью (основное + 2 мутации). Tier-ось не применяется к пользовательским генерациям: пользователь видит только визуальные мутации, а уровень исполнения выбирает позже — по описанию (без tier-специфичных превью).

**Характеристики:**
- 3 превью: основное + mutation_a + mutation_b
- Tier-специфичных визуальных превью нет
- На экране S9 пользователь выбирает уровень по описанию материалов и технологий
- Докрутка, ремиксы, шаринг — без изменений

## 4. Спецификация экранов (UI/UX Mobile-first)

### Общие принципы

- "Редакционный минимализм": мало шума, много воздуха, крупная типографика
- 1 главный CTA на экран
- Превью всегда в мокапе (обложка на блокноте)
- "В один тап": Докрутить, Поделиться, Заказать
- Share — ключ роста: кнопка всегда на видном месте

### Навигация (нижний tab bar)

1. **Каталог** — входные стили / коллекции / дропы
2. **Создать** — промпт + теги
3. **Поиск**
4. **Избранное**
5. **Профиль**

### S0. Splash / Preload

- Лого FORMA
- "Редакция вкуса. Твой блокнот."
- Индикатор прогресса
- **Fail:** кнопка "Повторить"

### S1. Каталог (главная витрина)

Секции (вертикальная лента):
- "Выбери, что нравится" (кнопка "Начать"/"Продолжить")
- Редакция FORMA (горизонтальный скролл)
- **Коллекции** (горизонтальный скролл карточек коллекций)
- Популярное сегодня
- **Лимитированные серии** (дропы с плашкой "Осталось X/30")
- Дроп / Коллекция недели
- Категории стилей

**Карточка дизайна:** мокап превью, название, 2-3 тега, метки ("Популярное", "Редакция", "Drop", "Осталось X")

**Карточка коллекции:** обложка коллекции, название, количество дизайнов, метка типа (regular / limited)

**Секция "Лимитированные серии":**
- Горизонтальный скролл карточек лимитированных коллекций/дропов
- Каждая карточка: обложка, название, плашка "Осталось X/30" (красная если X <= 5)
- Если X = 0 — плашка "Распродано", карточка полупрозрачная, не кликабельная
- Тап по карточке → S-DROP

### S2. Онбординг вкуса — "Выбери, что нравится"

- Свайп-карточки или сетка
- Header: "Выбери 5 стилей — и мы соберем твой вкус"
- Кнопки: "Не мое" / "Нравится"
- Прогресс: "Выбрано: 0/5"
- После 5 лайков → Bottom sheet: "Готово. Создадим блокнот в твоем стиле?"
- CTA: "Создать"

### S3. Создать (промпт + теги)

- Поле промпта (multiline)
- Placeholder: "Например: минималистичный морской стиль, серебро вместо золота..."
- Чипсы "Быстрые теги": Страна, Сезон, Стихия, Материал, Архитектура, Brand mood
- Блок "Основа стиля" (из онбординга)
- **CTA:** "Сгенерировать (Pro)" + "Будет 3 варианта"
- **Состояния:** GuestLimitWarning, DisabledCTA (пустой промпт)

### S4. Генерация (прогресс)

- "Собираем варианты"
- Шаги: "Собираем стиль" → "Готовим 3 варианта" → "Делаем превью"
- "Пока ждешь — посмотри популярное"
- **Timeout:** "Дольше обычного. Мы продолжим, а ты можешь вернуться"
- **Failed:** "Не получилось. Повторить" + "Изменить запрос"

### S5. Результат генерации — 3 превью (ключевой экран)

- Горизонтальная лента из 3 карточек (пейджинг)
- Карточка: мокап (крупно), плашка ("Основной" / "Мутация: заменили золото → серебро")
- Кнопки на карточке: "Выбрать" (primary), "Докрутить" (secondary)
- Панель: Сохранить, Поделиться, Ремикс, "Что изменили?"
- После "Выбрать" → CTA: "Продолжить → Наполнение"
- **OneFailed:** "Этот вариант не получился" + "Пересобрать"

### S6. Поделиться (share sheet)

- Превью (мини), "Поделиться дизайном"
- Переключатель: "Показывать теги на картинке" (OFF по умолчанию)
- "Скопировать ссылку", "Скачать изображение", Системный share

### S6b. Докрутка (Refine)

- "Быстрые замены" (чипсы): золото/серебро/хром/медь, минимализм/люкс/винтаж/техно...
- Теги по вкладкам: Страны, Сезоны, Стихии, Архитектура, Материалы, Brand mood
- Поле "Скажи словами"
- Чекбокс: "Сохранить композицию (рекомендуем)"
- **CTA:** "Обновить (3 превью)"
- **Состояния:** TooManyTags, PolicyWarning (brand mood)

### S7. Страница дизайна (публичная)

- Большое превью + галерея
- Название, плашки ("Редакция FORMA", "Популярное", "Drop", "Коллекция: ...")
- Теги (только публичные)
- Кнопки: "Ремикснуть" (primary), "Сделать похожий", "Заказать"
- Блоки: "Ремиксы", "Похожие по вкусу"
- Секция "FORMA DNA": "Собрано из: ...", дата, ID
- **Для каталожных дизайнов:** дополнительная секция "Уровни исполнения" — 3 мини-превью (Core / Signature / Lux) с подписями
- **Для лимитированных:** плашка "Лимитированная серия — осталось X/30"

### S8. Наполнение (внутренний блок)

- Список: Клетка, Линейка, Точки, Пустые, Планер (если в MVP)
- Мини-превью разворота
- Параметры: нумерация страниц, язык (RU/EN)
- **CTA:** "Дальше"

### S9. Уровень исполнения — Core / Signature / Lux

> Переименовано из "Комплектация Base / Pro / Elite" (v0.1)

3 карточки уровней исполнения:

**Для каталожных дизайнов (Путь A):** каждая карточка содержит визуальное превью дизайна в соответствующем уровне (Core / Signature / Lux). Пользователь видит реальную разницу — как выглядит обложка в бумажном исполнении, с soft-touch ламинацией или в натуральной коже.

**Для пользовательских дизайнов (Путь B):** карточки содержат описание материалов и технологий каждого уровня (без tier-специфичных визуальных превью). Пользователь выбирает по описанию.

#### Структура карточки уровня

| Элемент | Core | Signature | Lux |
|---------|------|-----------|-----|
| Превью (каталожные) | Мокап в Core | Мокап в Signature | Мокап в Lux |
| Превью (генерация) | Иконка уровня | Иконка уровня | Иконка уровня |
| Название | Core | Signature | Lux |
| Подзаголовок | "Красивая картинка" | "Чувствуешь разницу" | "Другая физика" |
| Обложка | Бумажная, ламинация | Soft-touch ламинация | Натуральная кожа |
| Технология | Плоская печать | Spot UV, слепое тиснение | Deep embossing, doming, металл |
| Цена | TBD | TBD | TBD |
| Метка | — | "Популярный выбор" (выделена по умолчанию) | "Коллекционный уровень" |

- Signature выделена по умолчанию
- **CTA:** "Оформить заказ"

- "Лимит генераций на сегодня исчерпан"

- Вторичная: "Войти", "Вернуться и отредактировать запрос"

### S-COL. Страница коллекции (новый экран)

Публичная страница коллекции — подборки дизайнов, объединенных темой.

**Структура (сверху вниз):**

1. **Header коллекции**
   - Обложка коллекции (полноширинная, высота ~40% экрана)
   - Название коллекции (крупная типографика поверх обложки, снизу)
   - Количество дизайнов: "12 дизайнов"
   - Для limited: плашка "Лимитированная серия — осталось X/30"

2. **Описание**
   - 2-3 предложения о коллекции (тон: редакционный, спокойный)
   - Теги коллекции (чипсы, кликабельные → поиск)

3. **Сетка дизайнов**
   - Grid 2 колонки (mobile), 3-4 (desktop)
   - Карточка: мокап превью, название, метка уровня (Core/Signature/Lux — для каталожных)
   - Тап по карточке → S7 (страница дизайна)

4. **Для лимитированных коллекций:**
   - Счетчик "Осталось X из N"
   - Если X <= 5 — красная плашка "Осталось мало"
   - Если X = 0 — "Распродано", кнопки заказа неактивны
   - Каждая карточка дизайна показывает, доступен ли он

5. **Секция "Похожие коллекции"**
   - Горизонтальный скролл коллекций той же тематики

**Состояния:**
- **Пустая коллекция:** "Скоро здесь появятся дизайны" (скрыта из каталога, доступна по прямой ссылке)
- **Распроданная (limited):** обложка с полупрозрачным оверлеем, "Тираж распродан", кнопка "Подписаться на похожие"

### S-DROP. Страница дропа (новый экран)

Лимитированный выпуск — showcase с нумерацией экземпляров. Отличается от обычной коллекции акцентом на эксклюзивность и ограниченность.

**Структура (сверху вниз):**

1. **Hero-секция**
   - Полноэкранная обложка дропа (темный/контрастный фон)
   - Название дропа (крупно)
   - Подзаголовок: "Лимитированная серия — N экземпляров"
   - Счетчик: "Осталось X из N" (визуально выделен)
   - CTA: "Смотреть коллекцию" (scroll down)

2. **О дропе**
   - Редакционный текст (история, вдохновение, идея)
   - "Каждый экземпляр пронумерован и уникален"

3. **Сетка дизайнов**
   - Grid 1-2 колонки (mobile, крупные карточки)
   - Карточка: мокап в уровне Lux (дропы всегда по цене Lux)
   - Номер экземпляра: "1/30", "2/30" и т.д. (для уже проданных — "Продан")
   - Тап по доступному → S7

4. **Детали исполнения**
   - Описание уникальных особенностей дропа:
     - Уникальный рисунок вставки на каждом экземпляре
     - Нумерация на шильде
     - Особый форзац и цвета лент
     - Паспорт коллекционера с подписью

5. **Уровни (для дропа — всегда Lux)**
   - Краткое описание уровня Lux
   - Цена
   - CTA: "Заказать" → S8

**Состояния:**
- **Предзаказ (до релиза):** обратный отсчет, кнопка "Напомнить мне"
- **Активный:** доступен для заказа, "Осталось X из N"
- **Последние экземпляры (X <= 5):** красная плашка "Осталось мало!", urgency-анимация
- **Распродан:** "Тираж распродан", галерея дизайнов без кнопки заказа, "Подписаться на следующий дроп"

## 5. Карта переходов

### 5.1. Глобальные правила

**CurrentDesign (черновик)** — всегда может существовать:
- выбранный стиль, промпт, теги, последний превью, наполнение, уровень
- Если пользователь уходит/возвращается — показываем "Продолжить"

**Гейт лимитов:** перед генерацией (S3, S6b) → проверка лимита → L1 если исчерпан

**Гейт оплаты (гость):** на S10 перед оплатой → требовать Auth (рекомендуется)

**Гейт наличия (limited):** перед оформлением заказа из лимитированной коллекции/дропа → проверка `stock_remaining > 0`. Если тираж распродан — показать "Распродано", предложить подписку на похожие.

## 6. Микрокопи FORMA (тональность)

- "Собираем варианты" (не "processing...")
- "Идея от редакции" (для мутаций)
- "Докрутить" (ключевое слово)
- "Сделать похожий"
- "Вкус сохранен"
- "Уровень исполнения" (не "комплектация" или "тариф")
- "Core / Signature / Lux" (не "базовый / продвинутый / элитный")
- "Осталось X из 30" (для лимитированных, не "в наличии")
- "Тираж распродан" (не "нет в наличии")
- "Экземпляр N/30" (не "номер заказа")
- "Лимитированная серия" (не "ограниченный тираж")

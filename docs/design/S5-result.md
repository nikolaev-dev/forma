# S5 — Результат генерации (3 превью)

> Экран для генерации в v0.dev
> Проект: FORMA — сервис генерации дизайна обложек блокнотов
> Платформа: Mobile-first (375px), адаптив на desktop

---

## Контекст

Пользователь задал промпт + теги, нажал "Сгенерировать", подождал 10-30 секунд на экране прогресса (S4) и попал сюда. Это **ключевой экран продукта** — момент, когда человек впервые видит свой дизайн.

На экране — 3 варианта обложки блокнота:
- **Основной** — точно по запросу
- **Мутация A** — система заменила 1-2 тега (например, "золото → серебро")
- **Мутация B** — другая замена (например, "минимализм → ар-деко")

Мутации — это "идеи от редакции FORMA", не ошибки. Они показывают пользователю неожиданные направления.

---

## Бренд и визуальный тон

- **Стиль:** редакционный минимализм — много воздуха, крупная типографика, спокойные цвета
- **Тон:** "Редакция вкуса", не "AI-генератор". Спокойный, умный, без технологического шума
- **Шрифт:** sans-serif, современный (Inter / Manrope / аналог)
- **Палитра:** светлый фон (#FAFAF8 или подобный тёплый off-white), тёмный текст, один акцентный цвет для CTA
- **Ощущение:** как листать журнал про дизайн, не как пользоваться AI-инструментом

## Структура экрана (сверху вниз)

### 1. Header (компактный)

- Кнопка "← Назад" (возврат к промпту / создать)
- Заголовок: **"Ваши варианты"** или **"3 варианта"**
- Иконка "..." (меню: сохранить все, новая генерация)

### 2. Горизонтальная лента карточек (главный блок)

Три карточки с горизонтальным пейджингом (snap scroll). Центральная карточка видна полностью, соседние — частично (подглядывают по краям на ~20px).

**Каждая карточка содержит:**

#### 2a. Мокап (основная визуальная область)
- Изображение обложки наложено на реалистичный мокап блокнота (блокнот лежит на поверхности, лёгкая тень)
- Соотношение сторон мокапа ~3:4
- Занимает ~65% высоты экрана
- Скруглённые углы карточки (radius 16-20px)

#### 2b. Плашка-лейбл (поверх мокапа, внизу)
- Для основного варианта: **"Основной"** — нейтральная плашка (тёмный полупрозрачный фон)
- Для мутаций: **"Идея редакции: золото → серебро"** — акцентная плашка
- Шрифт мелкий (12-13px), 1 строка, с эллипсисом если длинный текст

#### 2c. Кнопки на карточке (под мокапом, внутри карточки)
- **"Выбрать"** — primary кнопка (заполненная, акцентный цвет), полная ширина карточки
- **"Докрутить"** — text/ghost кнопка под ней, не выделяется визуально

### 3. Пейджинг-индикатор

- 3 точки под лентой карточек
- Активная точка — крупнее или другого цвета
- Позволяет понять, какой из 3 вариантов смотришь

### 4. Панель действий (нижняя)

Горизонтальная полоска иконок с подписями, фиксированная к низу или под индикатором:

| Иконка | Подпись | Действие |
|--------|---------|----------|
| Закладка | Сохранить | Добавить в избранное |
| Стрелка-шер | Поделиться | Открыть share sheet (S6) |
| Развилка | Ремикс | Перейти в докрутку (S6b) как fork |
| Инфо (i) | Что изменили? | Bottom sheet с деталями мутаций |

## Состояния экрана

### Состояние 1: Все 3 варианта готовы (основное)
Описано выше. Все карточки с изображениями.

### Состояние 2: После выбора варианта
- Выбранная карточка получает тонкую обводку или чекмарк в углу
- Кнопка "Выбрать" меняется на "Выбрано ✓" (disabled-стиль)
- Снизу появляется плавающий CTA-бар: **"Продолжить → Наполнение"** (ведёт на S8)
- Этот CTA — primary, полная ширина, с лёгкой тенью

### Состояние 3: Один вариант не получился (partial)
- Вместо изображения — заглушка: иконка + текст **"Этот вариант не получился"**
- Кнопка на карточке: **"Пересобрать"** вместо "Выбрать"
- Остальные 2 карточки работают нормально

### Состояние 4: Все варианты неудачные (edge case)
- Полноэкранное сообщение вместо ленты:
  - **"Не удалось собрать варианты"**
  - Кнопка: **"Попробовать снова"** (primary)
  - Кнопка: **"Изменить запрос"** (secondary, возврат на S3)

## Bottom sheet "Что изменили?"

Появляется по тапу на иконку (i) в панели действий:

```
┌─────────────────────────────────┐
│  Что изменили в мутациях        │
│                                 │
│  Вариант A                      │
│  золото → серебро               │
│  "Заменили материал для         │
│   более холодного звучания"     │
│                                 │
│  Вариант B                      │
│  минимализм → ар-деко           │
│  "Добавили орнаментальность     │
│   и ретро-детали"               │
│                                 │
│  ─────────────────              │
│  [Понятно]                      │
└─────────────────────────────────┘
```

## Микрокопи (тональность FORMA)

- Заголовок: "Ваши варианты" (не "Результаты генерации")
- Основной: "Основной" (не "Original")
- Мутации: "Идея редакции: ..." (не "Mutation: ...")
- Выбрать: "Выбрать" → после выбора: "Выбрано"
- Докрутить: "Докрутить" (не "Refine" или "Edit")
- CTA после выбора: "Продолжить → Наполнение"
- Ошибка: "Не получилось. Попробуем ещё раз?" (не "Error occurred")
- Тон: спокойный, дружелюбный, без восклицаний и капса

## Переходы

| Действие | Куда |
|----------|------|
| "← Назад" | S3 (создать) или S4 (если вернулся из истории) |
| "Выбрать" → "Продолжить" | S8 (наполнение) |
| "Докрутить" | S6b (докрутка — быстрые замены тегов) |
| "Поделиться" | S6 (share sheet) |
| "Ремикс" | S6b (как fork текущего дизайна) |
| "Сохранить" | Остаёмся на S5, тост "Сохранено в избранное" |
| "Пересобрать" (partial) | S4 (повторная генерация 1 варианта) |

## Адаптация Desktop (>768px)

- 3 карточки отображаются в ряд (без пейджинга)
- Карточки одинаковой ширины, max-width ~360px каждая
- Gap между карточками 24px
- Панель действий — справа от карточек или под ними в строку
- CTA "Продолжить" — по центру под карточками

---

## Технические заметки для дизайна

- Мокап блокнота — статичный шаблон с вырезом под обложку. Обложка вставляется программно
- Изображения будут загружаться прогрессивно: сначала blur-placeholder, потом полное изображение
- Snap scroll на мобильном: CSS `scroll-snap-type: x mandatory`
- Карточки должны быть достаточно высокими чтобы мокап читался без зума

## Примеры данных для прототипа

**Вариант 1 (Основной):**
- Обложка: абстрактная геометрия, тёплое золото, минимализм
- Лейбл: "Основной"

**Вариант 2 (Мутация A):**
- Обложка: та же композиция, но серебро вместо золота, холодные тона
- Лейбл: "Идея редакции: золото → серебро"

**Вариант 3 (Мутация B):**
- Обложка: орнаментальная версия, ар-деко элементы
- Лейбл: "Идея редакции: минимализм → ар-деко"
